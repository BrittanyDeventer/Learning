LoanerLaptops program README:

Loanerlaptops.jar is crawls the excel sheet and outputs.  (Code can be read from ReadXlsx.java)

LoanerLaptops.xlsx is the excel sheet tied to the Loaner Laptops Sharepoint list.

RunJava.ps1 is a powershell script that opens the excel sheet, refreshes the data so it's up to date, and runs the jar file.

LoanerLog.txt is the output/log file.



** This is not meant to be workable on github, it is simply for showcase purposes.  Most filepaths have been omitted. Thank you.
